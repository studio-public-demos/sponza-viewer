# Sponza Palace - 3D Monument Viewer

[![Live Demo](https://raw.githubusercontent.com/KhronosGroup/glTF-Sample-Models/main/2.0/Sponza/screenshot/large.jpg)](https://studio-public-demos.github.io/sponza-viewer/)

**[▶ Click here to open the 3D Viewer](https://studio-public-demos.github.io/sponza-viewer/)**

## Features

- 12 lighting presets (Dawn, Morning, Noon, Golden, Sunset, Twilight, Night, Moonlit, Overcast, Dramatic, Warm Interior, Cool Interior)
- Exposure, ambient, and direct light sliders with Shadow toggle
- Orbit controls (drag to rotate, scroll to zoom, right-drag to pan)
- First-person walk mode (W/A/S/D + mouse look) with pointer lock
- Wireframe toggle, auto-rotate, top-down view, fullscreen, reset view
- Mobile responsive

## Model

Sponza Atrium from the [Khronos glTF Sample Models](https://github.com/KhronosGroup/glTF-Sample-Models/tree/main/2.0/Sponza). Originally created by Frank Meinl (Crytek), based on Marko Dabrovic's original model.
