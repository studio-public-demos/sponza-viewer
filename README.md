## Technical Highlights
- **Engine:** Three.js with advanced PBR rendering.
- **Optimization:** Draco mesh compression for fast asset loading.
- **Interactivity:** Full orbit controls, FPS-style walkthrough, and dynamic environmental lighting.
- **Responsiveness:** Fully optimized for mobile and desktop viewing.

*Model: The Sponza Palace Atrium, donated to the public by Crytek for global illumination research.*
